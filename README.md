# todoapp-pwa-with-firebase

## Run to test Install PWA

```
npm run build
npx http-server dist
```

dan pilih dengan server `http://127.0.0.1:{port}`

atau deploy website berikut [Link Deploy]('https://todoapp-fd927.web.app/')
